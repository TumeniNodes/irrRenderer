Detailed Description

Structure holding all materials irrRenderer uses.

Member Data Documentation

irr::video::E_MATERIAL_TYPE irr::video::SMaterials::TransparentRef

Transparent alpha reference material.

Each texel is either fully transparent or solid. Supports lighting.
Works just as Irrlicht's irr::video::EMT_TRANSPARENT_ALPHA_CHANNEL_REF.

irr::video::E_MATERIAL_TYPE irr::video::SMaterials::Normal

Normal map.

Requires a normal map texture and a mesh with tangents.

irr::video::E_MATERIAL_TYPE irr::video::SMaterials::NormalAnimated

Normal map for animated meshes.

Requires a normal map texture, however does NOT require a tangent mesh, which means it can be easily used on Irrlicht's animated meshes.
Approximates the tangent in code, results in lower quality.

irr::video::E_MATERIAL_TYPE irr::video::SMaterials::Parallax

Parallax map.

Requires a normal map texture with the original height stored in the alpha channel. Plus a tangent mesh.
Param1 affects bumpiness.

irr::video::E_MATERIAL_TYPE irr::video::SMaterials::DetailMap

Detail map.

Meant as a replacement for Irrlicht's irr::video::EMT_DETAIL_MAP.
First texture is applied as a solid material. Then, it's mixed with the second texture, which is the detail map.
Param1 affects detail map repeat.
